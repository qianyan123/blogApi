git常用指令
1.git config --global user.name 设置本地用户
2.git config --global user.email 设置本地邮箱
3.git add <file> 添加文件到暂存区
4.git commit -m "" 将文件上传到分支
5.git status 查看当前分支状态
6.git diff HEAD -- <file> 查看工作区与分支上的区别
7.git log 查看commit的记录
8.git reflog  查看使用的命令日志
9.git reset --hard HEAD^ 查看上个版本
10. git restore <file>
